# 🛍️ Trenzy

> AI-Powered Product Discovery Platform

## Vision

Trenzy helps users discover products, overcome decision fatigue, and find relevant items through personalized recommendations. Instead of acting as a traditional e-commerce platform, Trenzy focuses on intelligent discovery and affiliate-based shopping.

---

# Problem Statement

Modern users are overwhelmed with choices.

Trenzy aims to simplify product discovery by combining:

* Search
* Recommendations
* Wishlist
* Recently viewed products
* Discover feeds

into one streamlined experience.
